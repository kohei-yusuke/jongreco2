# リリースノート — JongReco v1.0.0

**2026-07-20**
本番: https://jongreco2.vercel.app/ ・ 登録不要の計算ツール: https://jongreco2.vercel.app/calc

麻雀のスコアを、**正しく・気軽に・気持ちよく**記録できるように大きく作り直しました。

---

## ✨ 新機能

- **登録不要の点数計算ツール**
  アカウントなしで、その場で点数精算（ウマ・オカ・返し点・焼き鳥）ができます。複数半荘の累計・順位も自動集計。入力内容はお使いの端末に自動保存され、更新しても消えません。
- **はじめての方向けチュートリアル**
  初回に使い方をやさしくガイド。いつでも「使い方」からもう一度見られます（もちろんスキップ可）。
- **ウマ・オカの自由設定**
  ウマは順位ごとに、オカは「自動／手動」で自由に設定できます。
- **ダークモード**
  画面右上でライト／ダークを切り替えられます。
- **トップページを刷新**
  「登録せずに計算する」導線をわかりやすく配置しました。

## 🛠 主なバグ修正

- 画面によって**点数の計算結果が食い違う**問題を解消し、常に正しい精算結果を表示するようにしました。
- **順位・オカ・焼き鳥**の計算間違いを修正しました。
- 「スコア推移」と「累計得点」の**グラフが同じ内容**になっていた問題を修正しました。
- **対局が作成できない／対局履歴の点数がおかしい**問題を修正しました。
- 点数を登録しても**履歴やグラフがすぐに更新されない**問題を修正しました。

## 📱 使いやすさの改善

- **スマホでの点数入力を「百点棒」対応**に。`152` と打つだけで **15,200点**になります（毎回すべての桁を打つ必要がなくなりました）。
- スマホでも押しやすいボタン配置・見やすい配色に全面刷新しました。
- 入力の合計が配給原点と合っているかをその場でチェックできます。

## ℹ️ ご利用上のメモ

- 登録なしで使う計算ツールの記録は、**お使いの端末（ブラウザ）だけ**に保存されます。別の端末と共有したい・ずっと残したい場合は、無料登録をご利用ください。
